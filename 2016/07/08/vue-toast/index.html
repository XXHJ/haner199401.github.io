vue 文件中的export default

问题来了,那要怎么让页面实现上面的效果呢,在哪里添加定时器捏…
在vue文件中export default 会默认导出一个对象,之后经过 vue-loader进行加载(loader会帮你注册为vue中的组件并声明)
所以在 vue 中,多次引用组件就会出现重复的dom结构
